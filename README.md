# 骰评 — 用理论掷骰，为电影算命。

🎲 **骰评**是一个结构主义电影批评生成工具。输入电影名，掷出7颗被理论家「附身」的骰子——拉康、齐泽克、福柯、德勒兹……每次掷骰都是从数千万种组合中选取一个分析路径，生成一个有观点密度的电影解读。

它不是影评生成器。它是一台观点矩阵机器：同一部电影可以掷出截然不同的批判视角——有时从「马克思×阶级×规训」切入，有时从「拉康×欲望×移置」展开。每掷一次，就是在电影的结构空间中标记一个观察节点。

内置双轨阅读系统（Inside / Around），既可以对艺术片做文本分析，也可以对商业片做工业-文化诊断。55位导演的结构化风格档案自动影响骰子走向。支持本地模型（Ollama）或云端API（OpenAI / DeepSeek），模板引擎无需任何配置即可使用。

## 快速开始

1. 打开 https://你的用户名.github.io/theory-dice/
2. 输入电影名（如「寄生虫」）
3. 点击 🎲 掷
4. 模板引擎可以直接用——无需任何配置
5. 想用 AI 引擎？点 ⚙ → 选 OpenAI / DeepSeek / Ollama → 填 API Key

## 核心特性

### 7 颗分析骰子

| 骰子 | 功能 |
|---|---|
| 理论框架 | 12 位理论家（拉康/齐泽克/福柯/德勒兹……） |
| 核心问题 | 欲望/权力/阶级/身份/工业逻辑/平台算法…… |
| 分析对象 | 家庭/国家/空间/IP宇宙/社交媒体/算法推荐…… |
| 导演签名 | 场面调度/长镜头/蒙太奇——或具体导演的视觉签名 |
| 运作机制 | 规训/意识形态/拟像/情感工程/算法剪辑…… |
| 宏观语境 | 晚期资本主义/数字社会/新浪潮/新现实主义…… |
| 崩塌方向 | 主体性崩塌/意义危机/类型公式耗尽/IP循环失灵…… |

### 双轨阅读系统（Inside / Around）

- **Inside**：进入电影文本内部，分析空间/叙事/镜头语法
- **Around**：围绕电影的外部阅读，分析工业逻辑/观众合约/平台算法
- 自动混合模式：系统根据电影类型自动分配，也可以手动切换

### 55 位导演数据库

系统内置 55 位全球核心导演的结构化风格档案。AI 提取电影时自动识别导演，将导演的 theory_affinity 注入骰子加权，用导演签名词汇覆盖导演骰面。

### 双引擎架构

- **模板引擎**：本地模板拼接，零成本，即时出结果
- **AI 引擎**：调用 LLM API（OpenAI / DeepSeek / Ollama），按 6 条写作规则生成 250-400 字的结构化观点

### 写作规则

1. 第一句必须是不可替换的电影事实
2. 每句最多一个抽象概念
3. 理论必须后置（电影→观察→理论）
4. 骰子主导词贯穿全文
5. 收尾用问题不用总结
6. 阅读模式决定追问方向

## 运行要求

纯静态网页，部署在任何静态托管服务即可。需要：

- 现代浏览器（Chrome / Edge / Firefox / Safari）
- （可选）LLM API Key：OpenAI / DeepSeek / 任何兼容 OpenAI 格式的 API

### 本地运行

```bash
cd release
python -m http.server 8000
# 打开 http://localhost:8000
```

## 本地模型

支持 Ollama 作为本地推理引擎：

1. 安装 [Ollama](https://ollama.com/)
2. 拉取模型：`ollama pull hermes3:8b`
3. 打开骰评 → ⚙ → 选「Ollama（本地）」→ 选模型
4. 无需 API Key
